This year, API Days SF focuses on the API Design and Strategy supporting the Internet of Things revolution. Everyone is talking about IoT: Kickstarter is overflowing with new hardware/software projects, connected objects were everywhere at the last CES. But too little time is spent talking about the underlying architecture that supports these objects, and the real challenge these companies are facing when designing their API or getting developers to use it.
